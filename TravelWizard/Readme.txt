Project Brief

TravelWizard was designed to be a one stop location for people to booking pre-planned package holidays in one go. The website provides a series of package holidays for the customer to select from. In order to booking a customer must have an account set up with TravelWizard. Opening an account allows customers to book holidays and gives access to customer only webpages such as a mail box and a manage myaccount section. TravelWizard Admin side allows employees to have control over customer bookings and payments. Allows for customer queries to be answered, notifications to be sent to customer mailboxes and control over users.
